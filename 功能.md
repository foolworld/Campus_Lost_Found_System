### 5. 智能搜索 (/search) ⭐ 核心功能
实现方式 ：基于 TF-IDF + 余弦相似度 算法
 5.1 中文分词 ( app.py:54-55 )
```
def tokenize(text):
    return [word for word in jieba.lcut(text) 
    if word.strip()]
```
使用 jieba 库将中文文本切分为词语列表。
 5.2 TF计算 ( app.py:57-64 )
```
def compute_tf(tokens):
    tf = defaultdict(int)
    total = len(tokens)
    for token in tokens:
        tf[token] += 1
    for token in tf:
        tf[token] /= total
    return tf
```
词频 = 某词出现次数 / 总词数
 5.3 IDF计算 ( app.py:66-76 )
```
def compute_idf(documents):
    idf = {}
    total_docs = len(documents)
    for token in all_tokens:
        doc_count = sum(1 for doc in documents 
        if token in tokenize(doc))
        idf[token] = math.log((total_docs + 
        1) / (doc_count + 1)) + 1
    return idf
```
逆文档频率 = log((总文档数+1)/(包含该词的文档数+1)) + 1
 5.4 余弦相似度 ( app.py:86-93 )
```
def cosine_similarity(vec1, vec2):
    dot_product = sum(vec1.get(token, 0) * vec2.
    get(token, 0) for token in all_tokens)
    norm1 = math.sqrt(sum(v**2 for v in vec1.
    values()))
    norm2 = math.sqrt(sum(v**2 for v in vec2.
    values()))
    return dot_product / (norm1 * norm2)
```
计算两个向量的夹角余弦值，范围 [0, 1]，值越大越相似。
 5.5 搜索流程 ( app.py:95-116 )
1. 获取所有已审核的拾物信息
2. 将每条信息的 标题+描述+地点 拼接为文档
3. 计算所有文档的 IDF 值
4. 对搜索关键词计算 TF-IDF 向量
5. 对每条文档计算 TF-IDF 向量并与关键词向量计算余弦相似度
6. 按相似度排序，返回 Top-N 结果

## 五、安全机制
1. 密码安全 ：使用 Werkzeug 的 generate_password_hash() 哈希存储，不存储明文密码
2. Session安全 ：使用 Flask-Login 管理登录状态，防止伪造
3. 权限验证 ：关键操作前验证用户身份和权限
4. 文件上传 ：使用 secure_filename() 防止路径遍历攻击，限制文件大小（5MB）
5. 输入验证 ：注册时验证用户名和密码长度，确认密码一致性
